FPS Demo

A FPS demo made for a game engine design course

This project was created as a final for CIS 490 at K-State University. It was made with an old version of XNA. I don't know if it works with the new version, so the code is offered as is.

The goal was to show off a demo of a game engine that was created custom for the final project. I decided to do a first person shooter demo.
